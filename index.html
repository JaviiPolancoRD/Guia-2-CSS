Propiedades CSS

Descripción detallada de las propiedades CSS para estilos de texto, enlaces, imágenes, listas y tablas.

Tema 1 - Texto

Propiedad Valor Inicial Descripción
color Dependiente del navegador Define el color del texto. Se puede especificar con nombre de color, código hexadecimal, RGB, etc.
font-family Depende del navegador Especifica la fuente o conjunto de fuentes para el texto.
font-size medium Define el tamaño de la fuente. Puede ser en px, em, rem, %, etc.
font-weight normal Controla el grosor de la fuente, con valores como normal, bold, o numéricos (100–900).
font-style normal Define el estilo de la fuente, como normal, italic, o oblique.
font-variant normal Cambia la apariencia del texto a small-caps o normal.
font — Es un atajo para configurar varias propiedades de fuente en una sola línea.
text-align start Define la alineación horizontal del texto, como left, center, right, o justify.
line-height normal Controla el espacio entre líneas, puede ser numérico, porcentual, en px, em, etc.
text-decoration none Añade decoraciones al texto, como underline, overline, o line-through.
text-transform none Controla la capitalización del texto, como uppercase, lowercase, o capitalize.
vertical-align baseline Ajusta la posición vertical del texto en relación con su línea base, como top, middle, etc.
text-indent 0 Indica la sangría de la primera línea del texto en un bloque.
letter-spacing normal Define el espaciado entre letras, puede ser en px, em, etc.
word-spacing normal Define el espaciado entre palabras, puede ser en px, em, etc.
white-space normal Controla cómo se maneja el espacio en blanco y el salto de línea en el texto.

Tema 2 - Enlaces

Característica Propiedades Descripción
Tamaño, Color y Decoración font-size, color, text-decoration Ajusta el tamaño, color y decoración (por ejemplo, subrayado) del enlace.
Decoración Personalizada text-decoration, text-shadow Permite añadir sombras o efectos de subrayado, sobrelineado y tachado en enlaces.

Tema 3 - Imágenes

Característica Propiedades Descripción
Anchura y Altura width, height Define las dimensiones de la imagen.
Eliminar el Borde border Quita el borde de la imagen, útil para imágenes dentro de etiquetas <a>.

Tema 4 - Listas

Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador en listas, como circle, square, decimal, etc.
list-style-position outside Indica si el marcador de lista se coloca dentro o fuera del contenido.
list-style-image none Permite usar una imagen personalizada como marcador en las listas.
list-style — Es un atajo para configurar varias propiedades de estilo de lista en una sola línea.

Tema 5 - Tablas

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse table separate Determina si las celdas de la tabla tienen bordes separados o colapsados.
border-spacing table 0 Define el espacio entre bordes de celdas en tablas cuando border-collapse está en separate.
empty-cells table show Controla si se muestran o no las celdas vacías de una tabla.
caption-side table-caption top Indica la posición del título de la tabla, ya sea en la parte superior o inferior.
